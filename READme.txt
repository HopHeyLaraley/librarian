# Управление библиотекой

Данный проект представляет собой простую систему управления библиотекой. Он позволяет работать с коллекцией книг: добавлять, удалять, изменять их статус и искать книги по различным критериям.

## Возможности
- Просмотр списка всех книг.
- Добавление новой книги.
- Удаление книги по ID.
- Изменение статуса книги (в наличии/выдана).
- Поиск книги по названию, автору или году.
